🌸 Oasis Infobyte Internship Projects

A beautifully designed collection of Python mini-projects built during the Oasis Infobyte Internship, featuring GUI applications, automation tools, and AI-inspired interfaces with a Sakura neon aesthetic.

✨ Overview

This repository contains multiple Python-based projects developed to strengthen core programming skills, GUI design, and real-world application building. Each project focuses on simplicity, usability, and visual appeal using Tkinter and Python libraries.

All applications are styled with a Sakura-inspired UI theme 🌸 featuring soft colors, interactive elements, and smooth user experience.

🚀 How to Run

Clone the repository:

git clone https://github.com/divya2538/oasis-infobyte-internship.git
cd oasis-infobyte-internship

Run any project:

cd voice_assistant
python main.py

or

cd weather_app
python main.py
📦 Requirements

Install dependencies:

pip install speechrecognition pyttsx3 wikipedia requests

For microphone support:

pip install pyaudio
